# React Project

This is a React-based project using **Vite**, **TypeScript typings**, **Sass**, and **React Router**. It includes testing setup with **Vitest** and accessibility testing using **vitest-axe**.

## Project Info

Capstone project for Coursera's Meta Front-End Dev course.
Repo contains 
  - Wireframe and Design figma files
  - React + Vite project
  - gh-pages site preview

## Project Setup

1. Clone the repository with git clone:
2. Run -> npm install
3. Run tests with npx vitest
   - there should be 17 test files with 43 tests inside
4. To see the site either use npm run dev or go to -> "https://marcel-212.github.io/little-lemon/"

## Available Scripts

- npm run dev -> Starts the development server with Vite
- npm run build -> Builds the app for production
- npm run test -> Runs tests using Vitest
